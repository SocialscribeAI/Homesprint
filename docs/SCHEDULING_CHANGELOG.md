## Development Notes

### Implementation Decisions
- **Flexible Location Model**: Support for physical, virtual, and hybrid viewings
- **Status-Driven Workflow**: Clear state machine for viewing lifecycle
- **Calendar-First Approach**: ICS export as primary calendar integration method
- **Automated Communication**: Proactive reminders and status updates

- Bulk scheduling performance optimization
